## AgentCore Memory Store

引入了按用户保留 365 天对话历史的 Memory Store。存储/检索逻辑由 `src/lib/agentcore-memory.ts` 管理。

<MemoryStoreSequence />

| 项目 | 说明 |
|------|------|
| 保留期限 | 365 天（`eventExpiryDuration`） |
| 用户隔离 | 从 Cognito JWT 中提取 email（`src/lib/auth-utils.ts`） |
| 存储数据 | 问题、摘要、路由、使用的工具、响应时间 |
| 设置脚本 | `06e-setup-agentcore-memory.sh` |
| 配置文件 | `data/config.json` 中的 `memoryId`、`memoryName` |

:::warning Memory 命名规则
Memory 名称中**不能使用连字符（-）**，只允许使用下划线（_）。
正确示例：`awsops_memory` / 错误示例：`awsops-memory`
:::

**AgentCore 页面集成**：
- 6 个 StatsCard：总调用次数、平均响应时间、各 Gateway 调用占比、各模型 token 使用量
- 对话历史搜索：关键词、日期范围、按路由过滤

**AI 页面集成**：
- 在底部切换面板中查看对话历史
- 可引用之前的问题/回答上下文

## 新页面（4 个）

<NewPagesFlow />

| 页面 | 路径 | 查询文件 | 主要功能 |
|--------|------|-----------|-----------|
| **EBS** | `/ebs` | `queries/ebs.ts` | 卷/快照、加密、EC2 挂载、Idle 卷检测 |
| **MSK** | `/msk` | `queries/msk.ts` | Kafka 集群、Broker 节点指标（CPU/Memory/Network）、KRaft 模式 |
| **OpenSearch** | `/opensearch` | `queries/opensearch.ts` | 域、N2N/At-Rest 加密、VPC 配置、集群状态 |
| **Resource Inventory** | `/inventory` | `resource-inventory.ts` | 18 种资源数量趋势（基于快照，零额外查询） |

:::info Steampipe JSONB 注意事项
- **MSK**：`provisioned` 列为 JSONB，包含 Broker 数量、存储大小等
- **OpenSearch**：需检查 `encryption_at_rest_options` JSONB 中的 `enabled` 字段
- 编写查询前必须先通过 `information_schema.columns` 确认列名
:::

## 容器成本分析

新增了 2 个页面和 API，用于分析 ECS 和 EKS 容器成本。

| 项目 | ECS（`/container-cost`） | EKS（`/eks-container-cost`） |
|------|------------------------|-----------------------------|
| **数据源** | CloudWatch Container Insights | OpenCost API |
| **回退方案** | 无 | 基于 Request 的计算 |
| **成本计算** | 应用 Fargate vCPU/GB 价格 | Pod CPU/Memory 分配成本 |
| **API 路由** | `/api/container-cost` | `/api/eks-container-cost` |
| **设置** | - | `06f-setup-opencost.sh`（Prometheus + OpenCost） |

## AI 路由改进

| 项目 | Before | After |
|------|--------|-------|
| AgentCore 配置 | 代码内硬编码 | 从 `data/config.json` 读取 ARN |
| 工具使用展示 | 无 | 通过关键词匹配推断 MCP 工具 → UI 徽章 |
| 多路由失败 | 整体失败 | Bedrock Direct 回退 |
| 超时时间 | 60 秒 | **90 秒** |

:::info 工具使用推断
AgentCore 只返回最终文本（无 `tool_call` 标签）。`extractUsedTools()` 函数通过匹配响应内容中的关键词来推断所使用的 MCP 工具，并在 UI 中以徽章形式展示。
:::
