외부 관측성 스택을 데이터소스로 연동하고, **자연어를 쿼리로 변환**하는 기능과 **15섹션 AI 종합 진단** 리포트를 추가한 대형 릴리스입니다.

:::tip 핵심 변화
**7종 관측성 연동** · **자연어 → PromQL/LogQL/TraceQL/SQL** · **AI 라우팅 10→11 (`datasource`)** · **15섹션 종합 진단 + DOCX/PDF/스케줄** · **SSRF 방지**
:::

## 외부 관측성 통합 (7종)

AWS 데이터에 더해 기존 관측성 플랫폼을 데이터소스로 연동합니다.

| 종류 | 플랫폼 |
|------|--------|
| Metrics | Prometheus · Dynatrace · Datadog |
| Logs | Loki · ClickHouse |
| Traces | Tempo · Jaeger |

- `/datasources` — 데이터소스 CRUD · 연결 테스트 · 인증 구성
- `/datasources/explore` — 직접 쿼리 실행 + **AI 쿼리 생성(자연어 → PromQL/LogQL/TraceQL/SQL)**
- AI 어시스턴트의 `datasource` 라우트가 외부 메트릭 질문을 자동 분류 (AI 라우팅 **10 → 11**로 확장)

## AI 종합 진단 (15섹션)

- Bedrock Opus가 인프라 전반을 **15섹션**으로 자동 분석 — 비용/보안/안정성 중심
- **DOCX · Markdown · 브라우저 Print-to-PDF** 내보내기
- **주간/격주/월간** 자동 스케줄 (report scheduler)
- 인쇄용 리포트 페이지 `/ai-diagnosis/report` (A4 페이지 브레이크)

## 보안 (SSRF 방지)

- `datasource-client.ts`의 URL 검증이 AI 라우트 포함 모든 외부 fetch 경로를 보호
- allowlist 기반 사설망 접근 제어 + IPv6 사설/링크로컬(`fc00::/7`, `fe80::/10`) 탐지
- 데이터소스 쿼리·AI 쿼리 생성은 관리자 전용

## 기타

- 멀티 라우트 합성에 **Converse Stream API** + AgentCore 응답 타이핑 효과
- 좀비 PostgreSQL 커넥션 자동 정리, 모니터링 쿼리 시간 범위 필터로 pg pool 고갈 방지
- EKS Access Entry 상태 표시 + kubeconfig 등록, EKS 서비스 리소스 탭
