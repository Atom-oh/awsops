# Cost Explorer

이번 달 누적 비용과 서비스별 비용 분포를 분석할 수 있는 페이지입니다.

<Screenshot src="/screenshots/cost/cost-explorer.png" alt="Cost Explorer" />

## 주요 기능

### 핵심 지표 카드
페이지 상단에 5개의 지표 카드가 비용 현황을 요약합니다:
- **이번 달 누적**: 이번 달 1일부터 현재까지의 누적 비용
- **전월 대비 (MoM · 일평균)**: 전월 대비 증감률. 이번 달은 진행 중이므로 **일평균** 기준으로 비교하여 부분 집계로 인한 왜곡을 줄입니다
- **예상 월말 비용**: AWS 예측값 또는 선형 추정값 (카드 하단에 **AWS 예측** / **선형 추정** 표시)
- **서비스 수**: 비용이 발생한 서비스 개수
- **최대 서비스**: 가장 많은 비용이 발생한 서비스와 금액

### 추이 차트
- **월별 비용 추이**: 최근 약 6개월간의 월별 비용을 영역 차트로 표시
- **일별 비용 추이**: 최근 약 30일간의 일별 비용을 영역 차트로 표시

### 서비스별 분포
- **서비스별 비용**: 서비스별 비용을 가로 막대 목록으로 표시
- **비용 구성**: 상위 서비스와 나머지를 묶은 **기타** 항목을 도넛 차트로 표시
- **서비스 상세 테이블**: 서비스 / 비용 / 점유율 컬럼의 정렬 가능한 테이블

### 서비스 드릴다운 패널
테이블에서 서비스 행을 클릭하면 오른쪽에 상세 패널이 열립니다:
- **일별 비용 추이 (30일)**: 해당 서비스의 최근 30일 일별 비용
- **사용 유형 분해**: 최근 3개월간 사용 유형(Usage Type)별 비용 분해
- 패널 경계를 드래그하여 폭을 조절할 수 있고, **×** 버튼 · **Esc** · 바깥 영역 클릭으로 닫습니다

<Screenshot src="/screenshots/cost/cost-drilldown.png" alt="서비스 드릴다운 패널" />

## 사용 방법

1. 사이드바에서 **비용**을 클릭합니다
2. 상단 지표 카드에서 이번 달 누적 비용과 월말 예상치를 확인합니다
3. 추이 차트로 월별·일별 비용 흐름을 파악합니다
4. **서비스별 비용** 목록과 **비용 구성** 도넛으로 비용 분포를 확인합니다
5. **서비스 상세** 테이블에서 행을 클릭하여 해당 서비스의 일별 추이와 사용 유형 분해를 확인합니다
6. 새로고침 버튼으로 최신 데이터를 다시 불러옵니다

## 사용 팁

:::tip 부분 집계 이해
이번 달은 아직 진행 중이므로 **전월 대비** 지표는 전체 월 합계가 아닌 **일평균** 기준으로 비교합니다. 그래서 월 초·중순에도 신뢰할 수 있는 증감률을 보여줍니다.
:::

:::info 부분 실패 시 동작
추이·월별·예측 데이터 일부를 불러오지 못해도 서비스별 분포는 그대로 표시됩니다. 비용 데이터가 보이지 않으면 Cost Explorer 권한 또는 세션 상태를 확인하세요.
:::

:::info 시간 표기
화면에 표시되는 시각은 한국 표준시(KST, Asia/Seoul) 기준입니다.
:::
