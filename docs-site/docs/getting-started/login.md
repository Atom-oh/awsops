# 로그인

이메일과 비밀번호로 AWSops에 로그인할 수 있는 페이지입니다.

<Screenshot src="/screenshots/getting-started/login.png" alt="로그인 화면" />

## 주요 기능
### 로그인 폼
- **이메일**: 발급받은 이메일 주소를 입력합니다.
- **비밀번호**: 비밀번호를 입력합니다.
- **로그인 유지**: 체크하면 브라우저를 닫았다 다시 열어도 세션이 유지됩니다. 공용 PC에서는 해제하세요.
- **로그인 →**: 입력한 자격 증명으로 로그인을 시도하는 버튼입니다.

### 자동 리다이렉트
- 로그인하지 않은 상태로 아무 페이지에나 접근하면 자동으로 **로그인** 화면으로 이동합니다.
- 로그인에 성공하면 대시보드(또는 원래 가려던 페이지)로 곧바로 이동합니다.

### 오류 안내
- 잘못된 이메일·비밀번호, 추가 인증이 필요한 계정, 일시적인 서비스 오류는 폼 안에 **인라인 알림**으로 표시됩니다.

## 사용 방법
1. 브라우저에서 AWSops 주소로 접속합니다. 로그인하지 않았다면 **로그인** 화면이 나타납니다.
2. **이메일**과 **비밀번호**를 입력합니다.
3. 세션을 유지하려면 **로그인 유지**를 체크된 상태로 둡니다.
4. **로그인 →** 버튼을 클릭합니다.
5. 인증이 완료되면 대시보드로 이동합니다.
6. 로그아웃하려면 왼쪽 사이드바 하단의 내 계정 옆에 있는 **로그아웃 아이콘**을 클릭합니다. 세션이 정리되고 다시 **로그인** 화면으로 돌아갑니다.

## 사용 팁
:::tip 로그인 유지
**로그인 유지**를 체크하면 다음 접속 시 다시 로그인하지 않아도 됩니다. 공용·공유 기기에서는 체크를 해제하고 사용 후 반드시 로그아웃하세요.
:::

:::info 로그인이 안 될 때
폼 상단에 인라인 알림이 표시됩니다. 이메일·비밀번호를 다시 확인하고, 계속 실패하면 잠시 후 다시 시도하거나 관리자에게 문의하세요.
:::
