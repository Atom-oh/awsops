# ADR-031 Phase 1 Implementation Plan — Runtime-Customizable Agents & Skills (Catalog + Resolver)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let an admin create reusable **skills** and **agents** in the dashboard at runtime, compose them many-to-many, and have the AI chat use a per-request **resolved spec** — without a Docker rebuild — bound to the existing 125 MCP tools only.

**Architecture:** A Skill+Agent catalog lives in Aurora (ADR-030). A `src/lib` **resolver** turns a selected agent into an effective spec (`gateway` + composed `systemPromptOverride` + `toolAllowlist` + traceability). `route.ts` passes that spec in the AgentCore payload; `agent.py` becomes **registry-agnostic** — it renders `systemPromptOverride` when present, else falls back to the built-in `SKILL_BASE` path (unchanged for the 8 built-in gateways). A one-time `agent.py` change enables this; thereafter adding/editing custom skills needs no rebuild.

**Tech Stack:** Next.js 14 (App Router, TS), Aurora PostgreSQL via `pg` (`src/lib/db.ts`), Strands `agent.py` on AgentCore Runtime, vitest, Node `crypto` (SHA-256).

**Scope (Phase 1 only):** Catalog (skills/agents/agent_skills/audit) + admin CRUD API + admin UI + resolver + classifier candidate extension + registry-agnostic `agent.py` + SHA-256 integrity + `SKILLS_SOURCE=local` hot-reload + traceability logging + built-in catalog seeding. **Out of scope (later phases):** per-account Agent Spaces (Phase 2), BYO-MCP (Phase 3), cosign (Phase 3), mutating tools (Phase 4).

**Key contracts discovered (do not break):**
- `agent/agent.py` `handler(payload)` reads `payload.get("gateway")`, `payload.get("skill", gateway_role)`, `payload.get("accountId")`, `payload.get("accountAlias")`, `payload.get("messages")`. System prompt = `build_skill_prompt(skill_role, tools) + account_directive`, where `build_skill_prompt = SKILL_BASE[role] + tool_section + COMMON_FOOTER`.
- `src/app/api/ai/route.ts`: `ROUTE_REGISTRY: Record<string, RouteConfig>` (RouteConfig = `{ gateway, skill?, display, description, tools[], examples?, handler? }`); `invokeAgentCore(messages, gateway, accountId?, accountAlias?, skill?)` builds `payload = { messages, gateway, accountId, accountAlias, skill? }`.
- `src/lib/db.ts` exports `isAuroraEnabled()`, `getDb(): Promise<Pool>`, `checkDbHealth()`, `closeDb()`.
- `src/lib/auth-utils.ts` `getUserFromRequest(req)`; admin gate pattern (see `src/app/api/parity/route.ts`): empty `adminEmails` ⇒ everyone admin, else must be in list.
- `infra-cdk/data/schema.sql` ends with a `schema_migrations` insert (version 1) and `COMMIT;`.

**Tool catalog (known tool names for allowlist validation):** the resolver/validation use the union of `RouteConfig.tools` capability strings is NOT the tool id list. The authoritative tool-id list for Phase 1 is the 8 gateway keys (`network, container, iac, data, security, monitoring, cost, ops`) — a custom agent binds to exactly **one** gateway, and its `toolAllowlist` is validated to be a subset of that gateway's MCP tool names. Since the Next.js side does not have the live MCP tool list, Phase 1 validates `toolAllowlist` against a static `KNOWN_GATEWAYS` set and treats the per-tool allowlist as advisory metadata passed to `agent.py` (hard per-tool enforcement is Phase 2 once the gateway tool inventory is surfaced). This limitation is logged.

// src/lib/db/catalog.ts
// ADR-031 Phase 1 — Aurora-backed skill/agent catalog (CRUD + queries + SHA-256).
import { createHash } from 'crypto';
import { getDb, isAuroraEnabled } from '@/lib/db';

export type Tier = 'builtin' | 'custom';

export interface SkillInput {
  name: string;
  description: string;
  instructions: string;
  toolAllowlist: string[];
  tier: Tier;
  createdBy?: string;
}
export interface AgentInput {
  name: string;
  description: string;
  persona: string;
  routingKeywords: string[];
  gateway: string;
  model?: string;
  tier: Tier;
  createdBy?: string;
}
export interface AgentWithSkills {
  id: number; name: string; description: string; persona: string; gateway: string;
  tier: Tier; version: number; enabled: boolean; routingKeywords: string[];
  skills: Array<{ name: string; instructions: string; contentHash: string; ord: number; toolAllowlist: string[] }>;
}

/** SHA-256 over canonical JSON of the integrity-relevant fields. Order-independent on arrays. */
export function computeSkillHash(s: { name: string; description: string; instructions: string; toolAllowlist: string[] }): string {
  const canonical = JSON.stringify({
    name: s.name, description: s.description, instructions: s.instructions,
    toolAllowlist: [...s.toolAllowlist].sort(),
  });
  return createHash('sha256').update(canonical).digest('hex');
}

export async function upsertSkill(s: SkillInput): Promise<number> {
  const db = await getDb();
  const hash = computeSkillHash(s);
  const { rows } = await db.query(
    `INSERT INTO skills (name, description, instructions, tool_allowlist, tier, content_hash, created_by, enabled)
     VALUES ($1,$2,$3,$4::jsonb,$5,$6,$7, false)
     ON CONFLICT (name) DO UPDATE
       SET description = EXCLUDED.description,
           instructions = EXCLUDED.instructions,
           tool_allowlist = EXCLUDED.tool_allowlist,
           content_hash = EXCLUDED.content_hash,
           version = skills.version + 1,
           updated_at = NOW()
     RETURNING id`,
    [s.name, s.description, s.instructions, JSON.stringify(s.toolAllowlist), s.tier, hash, s.createdBy ?? null],
  );
  return rows[0].id;
}

export async function upsertAgent(a: AgentInput): Promise<number> {
  const db = await getDb();
  const { rows } = await db.query(
    `INSERT INTO agents (name, description, persona, routing_keywords, gateway, model, tier, created_by, enabled)
     VALUES ($1,$2,$3,$4::jsonb,$5,$6,$7,$8, false)
     ON CONFLICT (name) DO UPDATE
       SET description = EXCLUDED.description, persona = EXCLUDED.persona,
           routing_keywords = EXCLUDED.routing_keywords, gateway = EXCLUDED.gateway,
           model = EXCLUDED.model, version = agents.version + 1, updated_at = NOW()
     RETURNING id`,
    [a.name, a.description, a.persona, JSON.stringify(a.routingKeywords), a.gateway, a.model ?? null, a.tier, a.createdBy ?? null],
  );
  return rows[0].id;
}

export async function attachSkill(agentId: number, skillId: number, ord = 0): Promise<void> {
  const db = await getDb();
  await db.query(
    `INSERT INTO agent_skills (agent_id, skill_id, ord) VALUES ($1,$2,$3)
     ON CONFLICT (agent_id, skill_id) DO UPDATE SET ord = EXCLUDED.ord`,
    [agentId, skillId, ord],
  );
}

export async function setEnabled(kind: 'skill' | 'agent', id: number, enabled: boolean): Promise<void> {
  const db = await getDb();
  const table = kind === 'skill' ? 'skills' : 'agents';
  await db.query(`UPDATE ${table} SET enabled = $1, updated_at = NOW() WHERE id = $2`, [enabled, id]);
}

export async function listAgentsWithSkills(opts?: { enabledOnly?: boolean }): Promise<AgentWithSkills[]> {
  const db = await getDb();
  const where = opts?.enabledOnly ? 'WHERE a.enabled = true' : '';
  const { rows } = await db.query(
    `SELECT a.id, a.name, a.description, a.persona, a.gateway, a.tier, a.version, a.enabled,
            a.routing_keywords,
            COALESCE(json_agg(json_build_object(
              'name', s.name, 'instructions', s.instructions, 'content_hash', s.content_hash,
              'ord', ags.ord, 'tool_allowlist', s.tool_allowlist
            ) ORDER BY ags.ord) FILTER (WHERE s.id IS NOT NULL), '[]') AS skills
     FROM agents a
     LEFT JOIN agent_skills ags ON ags.agent_id = a.id
     LEFT JOIN skills s ON s.id = ags.skill_id AND s.enabled = true
     ${where}
     GROUP BY a.id`,
  );
  return rows.map((r: any) => ({
    id: r.id, name: r.name, description: r.description, persona: r.persona, gateway: r.gateway,
    tier: r.tier, version: r.version, enabled: r.enabled, routingKeywords: r.routing_keywords ?? [],
    skills: (r.skills ?? []).map((sk: any) => ({
      name: sk.name, instructions: sk.instructions, contentHash: sk.content_hash,
      ord: sk.ord, toolAllowlist: sk.tool_allowlist ?? [],
    })),
  }));
}

export async function writeAudit(a: { actor: string; action: string; objectType: string; objectId: string; beforeHash?: string; afterHash?: string }): Promise<void> {
  if (!isAuroraEnabled()) return;
  const db = await getDb();
  await db.query(
    `INSERT INTO customization_audit (actor, action, object_type, object_id, before_hash, after_hash)
     VALUES ($1,$2,$3,$4,$5,$6)`,
    [a.actor, a.action, a.objectType, a.objectId, a.beforeHash ?? null, a.afterHash ?? null],
  );
}
```

// src/lib/skill-validation.ts
// ADR-031 Phase 1 — pure validation for admin-authored skills/agents.
export const KNOWN_GATEWAYS = ['network', 'container', 'iac', 'data', 'security', 'monitoring', 'cost', 'ops'] as const;
const NAME_RE = /^[a-z0-9][a-z0-9-]{1,63}$/;
const MAX_INSTRUCTIONS = 50_000;
const MAX_PERSONA = 20_000;

export interface ValidationResult { ok: boolean; errors: string[]; }

export function validateSkill(s: { name: string; description: string; instructions: string; toolAllowlist: unknown }): ValidationResult {
  const errors: string[] = [];
  if (!NAME_RE.test(s.name)) errors.push('name must be kebab-case, 2-64 chars');
  if (!s.description?.trim()) errors.push('description is required');
  if (typeof s.instructions !== 'string' || s.instructions.length === 0) errors.push('instructions required');
  if ((s.instructions?.length ?? 0) > MAX_INSTRUCTIONS) errors.push(`instructions exceed ${MAX_INSTRUCTIONS} chars`);
  if (!Array.isArray(s.toolAllowlist) || !s.toolAllowlist.every((t) => typeof t === 'string')) errors.push('toolAllowlist must be string[]');
  return { ok: errors.length === 0, errors };
}

export function validateAgent(a: { name: string; description: string; persona: string; gateway: string; routingKeywords: unknown }): ValidationResult {
  const errors: string[] = [];
  if (!NAME_RE.test(a.name)) errors.push('name must be kebab-case, 2-64 chars');
  if (!a.description?.trim()) errors.push('description is required');
  if ((a.persona?.length ?? 0) > MAX_PERSONA) errors.push(`persona exceeds ${MAX_PERSONA} chars`);
  if (!(KNOWN_GATEWAYS as readonly string[]).includes(a.gateway)) errors.push(`gateway must be one of ${KNOWN_GATEWAYS.join(', ')}`);
  if (!Array.isArray(a.routingKeywords) || !a.routingKeywords.every((k) => typeof k === 'string')) errors.push('routingKeywords must be string[]');
  return { ok: errors.length === 0, errors };
}

// src/lib/agent-resolver.ts
// ADR-031 Phase 1 — turn a selected route/agent into an effective spec.
// Pure: caller injects the candidate custom agents (from the catalog source).
import type { AgentWithSkills } from '@/lib/db/catalog';

export interface ResolvedAgentSpec {
  tier: 'builtin' | 'custom';
  gateway: string;
  skill?: string;                 // built-in SKILL_BASE key (built-in path)
  systemPromptOverride?: string;  // composed persona + skill instructions (custom path)
  toolAllowlist?: string[];       // advisory in Phase 1
  agentName: string;
  agentVersion?: number;
  skillHashes: string[];
}

const KNOWN_GATEWAYS = ['network', 'container', 'iac', 'data', 'security', 'monitoring', 'cost', 'ops'];

/**
 * @param routeKey  the route/agent key the classifier selected
 * @param candidates enabled custom agents from the catalog
 */
export function resolveAgent(routeKey: string, candidates: AgentWithSkills[]): ResolvedAgentSpec {
  const custom = candidates.find((a) => a.name === routeKey && a.enabled && a.tier === 'custom');
  if (custom) {
    const ordered = [...custom.skills].sort((a, b) => a.ord - b.ord);
    const skillBlock = ordered.map((s) => s.instructions).filter(Boolean).join('\n\n');
    const systemPromptOverride = [custom.persona.trim(), skillBlock].filter(Boolean).join('\n\n');
    const toolAllowlist = Array.from(new Set(ordered.flatMap((s) => s.toolAllowlist)));
    return {
      tier: 'custom',
      gateway: custom.gateway,
      systemPromptOverride,
      toolAllowlist: toolAllowlist.length ? toolAllowlist : undefined,
      agentName: custom.name,
      agentVersion: custom.version,
      skillHashes: ordered.map((s) => s.contentHash),
    };
  }
  // Built-in passthrough: route key maps to a gateway (or itself if it is a gateway name).
  const gateway = KNOWN_GATEWAYS.includes(routeKey) ? routeKey : routeKey;
  return { tier: 'builtin', gateway, skill: routeKey, agentName: routeKey, skillHashes: [] };
}

// src/lib/catalog-source.ts
// ADR-031 Phase 1 — catalog source: Aurora (default) or local dir (SKILLS_SOURCE=local).
import { existsSync, readdirSync, readFileSync } from 'fs';
import { resolve, join } from 'path';
import { isAuroraEnabled } from '@/lib/db';
import { listAgentsWithSkills, type AgentWithSkills } from '@/lib/db/catalog';

const LOCAL_DIR = resolve(process.cwd(), 'data/skills');
let cache: { at: number; data: AgentWithSkills[] } | null = null;
const TTL_MS = 30_000;

export function _clearCacheForTests() { cache = null; }

function readLocal(): AgentWithSkills[] {
  if (!existsSync(LOCAL_DIR)) return [];
  const out: AgentWithSkills[] = [];
  for (const f of readdirSync(LOCAL_DIR)) {
    if (!f.endsWith('.json')) continue;
    try {
      const a = JSON.parse(readFileSync(join(LOCAL_DIR, f), 'utf-8'));
      if (a && a.enabled && a.tier === 'custom') out.push(a);
    } catch { /* skip malformed local file */ }
  }
  return out;
}

export async function getEnabledCustomAgents(): Promise<AgentWithSkills[]> {
  if (process.env.SKILLS_SOURCE === 'local') return readLocal(); // hot-reload, no cache
  if (!isAuroraEnabled()) return [];
  const now = Date.now ? Date.now() : 0; // Date.now available in Node runtime
  if (cache && now - cache.at < TTL_MS) return cache.data;
  const data = await listAgentsWithSkills({ enabledOnly: true });
  cache = { at: now, data };
  return data;
}

// src/app/api/customization/route.ts
// ADR-031 Phase 1 — admin-gated CRUD for the skill/agent catalog.
import { NextRequest, NextResponse } from 'next/server';
import { getUserFromRequest } from '@/lib/auth-utils';
import { getConfig } from '@/lib/app-config';
import { isAuroraEnabled } from '@/lib/db';
import { validateSkill, validateAgent } from '@/lib/skill-validation';
import {
  upsertSkill, upsertAgent, attachSkill, setEnabled, listAgentsWithSkills, writeAudit,
} from '@/lib/db/catalog';

function isAdmin(req: NextRequest): { ok: boolean; email: string } {
  const user = getUserFromRequest(req);
  const config = getConfig();
  if (!config.adminEmails || config.adminEmails.length === 0) return { ok: true, email: user.email };
  return { ok: config.adminEmails.includes(user.email), email: user.email };
}

export async function GET(req: NextRequest) {
  if (!isAdmin(req).ok) return NextResponse.json({ error: 'Admin access required' }, { status: 403 });
  if (!isAuroraEnabled()) return NextResponse.json({ auroraEnabled: false, agents: [] });
  return NextResponse.json({ auroraEnabled: true, agents: await listAgentsWithSkills() });
}

export async function POST(req: NextRequest) {
  const admin = isAdmin(req);
  if (!admin.ok) return NextResponse.json({ error: 'Admin access required' }, { status: 403 });
  if (!isAuroraEnabled()) return NextResponse.json({ error: 'Aurora not configured' }, { status: 400 });
  const body = await req.json();
  if (body.kind === 'skill') {
    const v = validateSkill(body);
    if (!v.ok) return NextResponse.json({ error: 'Invalid skill', detail: v.errors }, { status: 400 });
    const id = await upsertSkill({ ...body, tier: 'custom', createdBy: admin.email });
    await writeAudit({ actor: admin.email, action: 'upsert', objectType: 'skill', objectId: String(id) });
    return NextResponse.json({ ok: true, id });
  }
  if (body.kind === 'agent') {
    const v = validateAgent(body);
    if (!v.ok) return NextResponse.json({ error: 'Invalid agent', detail: v.errors }, { status: 400 });
    const id = await upsertAgent({ ...body, tier: 'custom', createdBy: admin.email });
    await writeAudit({ actor: admin.email, action: 'upsert', objectType: 'agent', objectId: String(id) });
    return NextResponse.json({ ok: true, id });
  }
  return NextResponse.json({ error: 'kind must be skill|agent' }, { status: 400 });
}

export async function PUT(req: NextRequest) {
  const admin = isAdmin(req);
  if (!admin.ok) return NextResponse.json({ error: 'Admin access required' }, { status: 403 });
  const body = await req.json();
  if (body.op === 'enable' || body.op === 'disable') {
    await setEnabled(body.kind, body.id, body.op === 'enable');
    await writeAudit({ actor: admin.email, action: body.op, objectType: body.kind, objectId: String(body.id) });
    return NextResponse.json({ ok: true });
  }
  if (body.op === 'attach') {
    await attachSkill(body.agentId, body.skillId, body.ord ?? 0);
    await writeAudit({ actor: admin.email, action: 'attach', objectType: 'agent_skill', objectId: `${body.agentId}:${body.skillId}` });
    return NextResponse.json({ ok: true });
  }
  return NextResponse.json({ error: 'unknown op' }, { status: 400 });
}
```
