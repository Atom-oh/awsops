# 대시보드

AWS와 Kubernetes 운영 현황을 한눈에 살펴보고, AI 어시스턴트로 바로 이어갈 수 있는 메인 페이지입니다.

<Screenshot src="/screenshots/overview/dashboard.png" alt="대시보드" />

## 주요 기능

### AI Operations

페이지 상단의 **AI Operations** 행에서 AI 기능으로 곧장 진입할 수 있습니다.

- **AI Assistant 카드**: 활성 에이전트 수 배지가 표시되며, **대화 시작**을 누르면 채팅 드로어가 열립니다.
- **최근 AI 대화**: 지난 대화 목록을 보여 주며, 항목을 클릭하면 해당 대화를 다시 엽니다.
- **AI 분석** 상태 카드: **EKS 진단**/**K8sGPT** 게이트, 인시던트 자동 분석, 하이브리드 라우팅 정확도 등 현재 분석 환경 상태를 요약합니다.

### KPI 타일

핵심 리소스 지표를 세 그룹으로 나누어 표시합니다.

| 그룹 | 포함 지표 |
|------|-----------|
| **COMPUTE & CONTAINERS** | EC2(running/stopped 힌트), Lambda, ECS, ECR, EKS |
| **STORAGE & NETWORK** | S3, EBS(미암호화 볼륨 수), RDS, DynamoDB, VPC |
| **SECURITY · OPS · COST** | IAM 역할, Security Group(개방 인그레스 수), 작업 성공/실패, CloudWatch 알람, 이번 달 비용(USD) |

- 미암호화 볼륨, 0.0.0.0/0 개방 인그레스, 실패한 작업이 있으면 해당 타일이 **경고/위험** 색으로 강조됩니다.
- 이번 달 비용은 USD 금액으로 표시됩니다.

### 차트

| 차트 | 내용 |
|------|------|
| **리소스 분포** | 상위 리소스 유형별 개수(막대) |
| **카테고리별 리소스** | 카테고리별 비중과 총합(도넛) |
| **작업 상태** | 성공·실패·실행·대기 작업 비중(도넛) |
| **일별 비용 추이** | 날짜별 비용 추이(영역) |

## 사용 방법

1. 로그인 후 자동으로 대시보드가 열리며, 사이드바에서 **개요 > 대시보드**로도 이동할 수 있습니다.
2. **AI Operations** 행에서 **대화 시작**을 눌러 어시스턴트와 대화를 시작하거나, **최근 AI 대화**에서 이전 대화를 다시 엽니다.
3. KPI 타일에서 경고/위험 색으로 강조된 항목을 확인합니다.
4. 차트로 리소스 구성, 작업 상태, 비용 추이를 살펴봅니다.
5. 헤더의 **Refresh** 버튼으로 전체 데이터를 다시 불러옵니다. 마지막 갱신 시각이 함께 표시됩니다.

:::tip 최신 데이터 유지
**Refresh** 버튼은 마지막으로 데이터를 불러온 시각(KST)을 표시하며, 30분이 지나면 **(오래됨)** 표시가 붙습니다. 강조 타일이 보이거나 표시가 오래됐다면 한 번 새로 고쳐 주세요.
:::

:::info 데이터가 비어 보일 때
요약 데이터가 도착하기 전에는 KPI 타일이 **—** 로 표시됩니다. 작업이나 비용 데이터가 없을 때는 차트 자리에 **작업 데이터 없음** 또는 **비용 데이터 없음** 메시지가 표시됩니다.
:::

## AI 분석 팁

대시보드에서 눈에 띈 항목을 AI 어시스턴트에 그대로 물어볼 수 있습니다.

- "개방된 인그레스가 있는 Security Group이 어떤 것들인지 정리해 줘"
- "미암호화 EBS 볼륨 목록과 어떤 인스턴스에 붙어 있는지 알려 줘"
- "이번 달 비용이 늘어난 주된 원인이 무엇인지 분석해 줘"
- "최근 실패한 작업이 무엇이고 왜 실패했는지 진단해 줘"
