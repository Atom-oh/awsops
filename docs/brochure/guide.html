43개 운영 페이지, 전부 읽기 전용. 라이브 모니터링부터 토폴로지 그래프, AI 진단 리포트, 자연어 데이터소스 탐색까지 — 운영 의사결정에 필요한 컨텍스트를 한 곳에서 봅니다.

둘러보기 & 네비게이션

AWSops는 43개 운영 페이지로 구성된 읽기 전용 대시보드입니다. 리소스 모니터링, 인벤토리, 비용, EKS, 토폴로지, AI 진단, 어시스턴트, 데이터소스 탐색까지 운영 관점에서 자주 필요한 화면을 모았습니다.

◆

전부 읽기 전용입니다. AWSops는 AWS 리소스를 변경하지 않습니다. 진단과 분석은 제공하되, 실제 조치(remediation)는 운영자의 몫으로 남깁니다.

로그인 인앱 로그인 테마 teal / Cobalt / dark console 화면 모바일 반응형 계정 멀티계정 전환·통합 뷰

EKS — 온보딩 + 인클러스터 조회

클러스터를 온보딩하면 웹 BFF에서 직접 인클러스터 리소스를 읽습니다.

  1. 클러스터에 Access Entry + AmazonEKSViewPolicy를 부여해 온보딩합니다.
  2. 태스크 롤의 presigned-STS 토큰으로 BFF가 인클러스터에 직접 접속합니다.
  3. 노드 · 파드 · 디플로이먼트 · 서비스를 읽기 전용으로 조회합니다.
◆

K8sGPT 스타일 진단 레이어(ADR-035)도 읽기 전용으로 제공됩니다. 클러스터 상태를 진단하되 변경하지 않습니다.

토폴로지 그래프
