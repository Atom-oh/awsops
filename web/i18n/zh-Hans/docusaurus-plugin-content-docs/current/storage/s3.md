# S3

管理 S3(Simple Storage Service) 存储桶并监控其安全状态。

<Screenshot src="/screenshots/storage/s3.png" alt="S3" />

## 主要功能

### 统计卡片
- **Total Buckets**: 全部存储桶数量
- **Public Buckets**: 允许公开访问的存储桶数量（安全警告）
- **Versioning**: 已启用版本控制的存储桶数量
- **Logging**: 已配置访问日志的存储桶数量

### TreeMap 可视化
按区域直观展示存储桶：
- **红色**: Public 存储桶（需要注意）
- **绿色**: 已启用 Versioning 的存储桶
- **青色**: 普通存储桶

点击存储桶方块可跳转到详情面板。

### 可视化图表
- **Buckets by Region**: 按区域的存储桶分布
- **Security Status**: Private/Public/Versioned/Logging 状态分布

### 筛选
- 搜索框：按存储桶名称搜索
- 区域筛选：仅查看特定区域
- 访问筛选：仅查看 Public/Private 存储桶

### 详情面板
点击存储桶后可查看的信息：
- 存储桶名称、区域、ARN、创建日期
- 安全设置（Public Policy、Block ACLs 等）
- 版本控制、加密、生命周期规则
- 拥有 S3 访问权限的 IAM 角色列表
- 标签信息

## 使用方法

### 查看存储桶列表
1. 在 TreeMap 中查看按区域的存储桶分布
2. 在表格中查看详细列表
3. 利用筛选器搜索所需的存储桶

### 检查安全状态
1. 在 Public Buckets 卡片中确认公开存储桶数量
2. 在 TreeMap 中识别红色存储桶
3. 在访问筛选中选择 "Public" 查看列表

### 检查 IAM 权限
在存储桶详情面板的 "IAM Roles with S3 Access" 部分，可以查看能够访问该存储桶的 IAM 角色。

## 使用技巧

:::tip 注意公开存储桶
公开存储桶存在数据泄露风险。请定期检查是否存在非预期的公开存储桶。当 Public Buckets 卡片显示为红色时，需要立即进行审查。
:::

:::info 建议启用版本控制
对于存储重要数据的存储桶，请启用版本控制以防止误删除或覆盖。未启用版本控制的存储桶数量可在 Versioning 卡片中查看。
:::

## AI 分析技巧

可以向 AI 助手提出如下问题：

- "列出公开的 S3 存储桶并分析安全风险"
- "哪些存储桶未启用版本控制？"
- "分析可以访问特定存储桶的 IAM 角色"
- "检查 S3 存储桶的加密状态"

:::tip Data Gateway
AI 助手通过 Data Gateway 支持 S3 存储桶分析、安全检查、IAM 策略模拟等功能。还可以与 Security Gateway 联动进行深入的安全分析。
:::
