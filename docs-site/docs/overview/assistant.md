# AI 어시스턴트

자연어로 AWS·Kubernetes 운영 질문을 던지고 AI 답변을 받는 전체 화면 채팅 페이지입니다.

<Screenshot src="/screenshots/overview/assistant.png" alt="AI 어시스턴트 — 추천 질문" />

## 주요 기능

### 자연어 질문과 스트리밍 답변
- 평소 쓰는 말로 질문하면 답변이 **토큰 단위로 실시간 스트리밍**됩니다.
- 답변은 **마크다운**으로 렌더링되어 표, 코드 블록, 목록이 보기 좋게 표시됩니다.
- AWS 리소스는 **읽기 전용**으로 조회·분석만 하며, 리소스를 변경하지 않습니다.

<Screenshot src="/screenshots/overview/assistant-answer.png" alt="AI 어시스턴트 답변 (도메인 배지 + 마크다운 표)" />

### 자동 도메인 라우팅
- 질문은 가장 알맞은 도메인(**Network** · **Data** · **Security** · **Cost** · **Monitoring**)으로 **자동 분류**됩니다.
- 어떤 도메인이 답했는지는 답변에 표시되는 **색상 도메인 배지**(예: 💰 **Cost**)로 확인할 수 있습니다.
- 답변이 끝나면 같은 질문을 다른 도메인에 다시 물어볼 수 있는 **연관 도메인 칩**이 최대 2개 제시됩니다.

### 대화 목록 (왼쪽 레일)
- 왼쪽 레일에 저장된 대화가 **최신순**으로 쌓입니다.
- 대화를 클릭하면 다시 열 수 있고, 항목 위에 마우스를 올리면 **삭제** 버튼이 나타납니다.
- **새 대화** 버튼으로 빈 대화를 시작합니다.
- 대화는 자동 저장되며, 화면 어디서나 띄울 수 있는 **떠 있는 채팅 창**과 **같은 기록을 공유**합니다.

## 사용 방법
1. 사이드바에서 **어시스턴트**를 클릭하거나, 어느 페이지에서나 떠 있는 채팅 버튼을 엽니다.
2. 처음 화면의 **추천 질문 칩**(예: *이번 달 비용 추세*, *두 리소스 간 통신 원인*, *IAM 과다권한 점검*, *최근 알람 요약*) 중 하나를 누르거나 직접 입력합니다.
3. 답변이 스트리밍되는 동안 **도메인 배지**로 어느 도메인이 처리 중인지 확인합니다.
4. 답변 아래의 **연관 도메인 칩**을 눌러 같은 질문을 다른 도메인에서 다시 물어봅니다.
5. 왼쪽 레일에서 이전 대화를 클릭해 다시 열거나, **새 대화**로 새로운 주제를 시작합니다.

## 사용 팁

:::tip 특정 도메인으로 보내기
메시지 맨 앞에 **`/`**를 붙이면 해당 메시지 한 건을 특정 도메인으로 보낼 수 있습니다(예: `/network 두 서브넷 간 통신 확인`). 아직 **준비중**으로 표시된 도메인도 있습니다.
:::

:::info 떠 있는 채팅과 같은 기록
이 페이지의 대화는 떠 있는 채팅 창과 **동일한 기록**을 사용합니다. 특정 대화의 링크 끝에 `?thread=<id>`가 붙어 있으면 그 대화가 바로 열립니다.
:::

:::info 시간 표기
화면에 표시되는 시각은 한국 표준시(KST, Asia/Seoul) 기준입니다.
:::
